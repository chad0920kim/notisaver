# 개인정보처리방침 (Privacy Policy)

본 개인정보처리방침은 **알림 보관함 (NotiSaver)** 앱(이하 '앱')이 사용자의 개인정보를 어떻게 취급하는지 안내하기 위해 작성되었습니다.

본 앱은 사용자의 사생활을 소중히 여겨 가며, **완전 오프라인 우선(Offline-first)** 방식으로 작동합니다. 즉, 수집된 모든 알림 데이터는 외부 서버로 전송되지 않고 사용자의 기기 내부에만 안전하게 저장됩니다.

---

### 1. 처리하는 개인정보의 항목
본 앱은 Android의 알림 접근 권한(Notification Listener Service)을 사용하여 디바이스에 수신되는 푸시 알림 중 아래 항목을 기기 내부에 기록합니다.
* 수신된 앱의 패키지 이름 (예: `com.kakao.talk`)
* 알림의 제목 (Title)
* 알림의 내용 (Content/Text)
* 알림이 수신된 일시 (Timestamp)

### 2. 개인정보의 처리 목적
수집된 데이터는 오직 사용자가 과거에 수신한 알림 내역을 조회하고, 검색하고, 앱별로 필터링하여 관리하는 **앱 고유의 기능 제공 목적**으로만 사용됩니다.

### 3. 개인정보의 보유 및 이용 기간
* **로컬 저장 방식**: 수집된 모든 알림 로그는 오직 사용자의 기기 내부 로컬 데이터베이스(SQLite)에만 저장됩니다. 외부 서버나 제3자에게 전송되지 않습니다.
* **보유 기간**: 사용자가 앱 내에서 개별 알림을 삭제하거나, 설정에서 전체 로그 삭제를 수행하기 전까지 보존됩니다.
* **영구 파기**: 사용자가 디바이스에서 앱을 삭제(언인스톨)하는 경우, OS에 의해 기기 내부 로컬 데이터베이스도 즉시 영구 파기됩니다.

### 4. 개인정보의 제3자 제공 및 위탁
본 앱은 어떠한 경우에도 사용자의 개인정보를 제3자에게 제공하거나 외부에 위탁하지 않습니다. 네트워크 전송 API 자체가 포함되어 있지 않은 완전한 로컬 서비스입니다.

### 5. 정보주체(사용자)의 권리 및 행사방법
사용자는 언제든지 앱 내부에서 다음과 같은 권리를 행사할 수 있습니다.
* **개별 삭제**: 알림 목록에서 슬라이드(Dismissible) 제스처를 통해 개별 알림을 즉시 삭제할 수 있습니다.
* **전체 삭제**: 우측 상단 메뉴의 '전체 삭제(쓰레기통 아이콘)' 버튼을 눌러 로컬 DB에 보관된 모든 기록을 일괄 파기할 수 있습니다.
